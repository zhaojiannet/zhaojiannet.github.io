解决WP No Category Base插件卸载后出现404无法找到页面

发表于 2011-08-07 | 更新于 2017-11-07 | 分类于 技术随笔

因为需要去除wordpress的Category，所以安装了一个WP No Category Base插件。

但因为使用WP No Category Base插件后与我的博客自身的问题起冲突，所以卸载了。

卸载后wordpress博客所有页面出现了404错误，无法找到页面。

于是谷歌百度都未找到解决办法，下载了WP No Category Base插件的代码看了下，并没有更改数据库。

简单的推测并证实，解决办法非常简单：

1、记住或保存wordpress现有的固定链接方案。

2、更改wordpress固定链接为系统默认的任何一个。

3、改回wordpress固定链接为自己原设置的方案。

ok，一切正常了。

人生最难的不是奋斗，而是抉择。

人生最难的不是努力也不是奋斗，而是抉择！

当一切就像重新回到原点的时候，我并不认为我失去了所有，也并不认为我需要重新开始。我始终认为我只是到了下一个必经的十字路口，而在这个路口即将面临抉择十分重要。

这么多年我努力过，也奋斗过，可是到头来日子还是不好过，总是会发生这样那样的事，不是让我心情烦躁，就是工作不顺心。到现在，我终于明白，就像那句我经常会对别人讲的：“如果，舍不得、放不下，那就痛苦吧。”而我最近也越来越发现，放不下只是其中的一个原因，而更多的因素来自对生活的依赖，以及对工作的惰性。

只有倒光了第一杯水，才能装得下第二杯水。有些时候想想因为对梦想的贪恋，让自己失去的太对太多。而多少可以改变现状的机遇与我擦身而过的时候，都没有去珍惜。

梦想容易，创业难。

蝴蝶效应：1％的错误导致100％的失败

什么是蝴蝶效应？

1979年12月，洛伦兹在华盛顿的美国科学促进会的一次讲演中提出：一只蝴蝶在巴西煽动翅膀，有可能在美国的得克萨斯州引起一场龙卷风。其原因在于：蝴蝶翅膀的运动，导致其身边的空气系统发生变化，并引起微弱气流的产生，而微弱气流的产生又会引起它四周空气或其他系统产生相应的变化，由此引起连锁反应，最终导致其他系统的极大变化。这就是著名的“蝴蝶效应”。

从此以后，所谓“蝴蝶效应”之说就不胫而走，名声远扬了。

丢失一个钉子，坏了一只蹄铁；

坏了一只蹄铁，折了一匹战马；

折了一匹战马，伤了一位骑士；

伤了一位骑士，输了一场战斗；

输了一场战斗，亡了一个帝国。

马蹄铁上一个钉子是否会丢失，本是初始条件的十分微小的变化，但其“长期”效应却是一个帝国存与亡的根本差别。

在Windows XP下安装 SQL SERVER 2000 企业版

发表于 2011-05-25 | 更新于 2017-11-08 | 分类于 技术随笔

SQL2000企业版本适用于WIN 2000系统，2003系统和XP一般装不了需要选用个人版，当然如果你在不清楚的前提下辛辛苦苦下载了企业版本却不能安装，是不是很失望呢？这里介绍一个XP下安装装SQL2000企业版本方法以供参考~

办法如下：

一．在SQL服务器的安装盘中找到MSDE这个目录，并且点击setup.exe安装它，过程简单直接下一步就OK了。

二. 重启系统WINDOWSXP,这下就可以看到SQL服务的图标出现了。

三. 再拿出SQL服务器版的安装光盘，直接安装客户端工具（这个不要多说吧？最简单的方法就是直接点击光盘根目录下的autorun.exe)

根据提示安装，自检过程中知道系统不是SERVER版，会提示只安装客户端工具。（哈哈，服务端我已有了）

四. 打开企业管理器，试用SA用户连一下看看，是不是发现SA用户登陆失败？因为你还没有与信任SQL SERVER连接相关联。还好这个只要对系统注册表稍加修改就可以啦：

在运行中输入regedit打开注册表编辑器，找到

，这个项里面有一个键值LoginMode，默认下，值是1，现在将值改为2，重启电脑。

五. 再打开企业管理，再连接试试，是不是OK了？

解决傲游3打开网页时自动下载

发表于 2011-04-22 | 更新于 2017-11-08 | 分类于 技术随笔

解决傲游3打开网页时自动下载，傲游3做的非常失败，有时候打开网页却自动调用傲游下载下载html网页

改问题是由于傲游的下载功能引起的，默认只能通过傲游下载或者其他下载工具下载，不能ie下载，加上由于傲游webkit内核对web网页文件头和文件类型的识别上，出现bug导致

临时解放方案：

在傲游3的安装目录
Maxthon3\Bin 下删除 MxDownloader.exe ，删掉这个程序就可以恢复IE下载，或者你自己关联的下载。

10 个最酷的 Linux 单行命令

发表于 2011-04-09 | 更新于 2017-11-08 | 分类于 技术随笔

下面是来自 Commandlinefu 网站由用户投票决出的 10 个最酷的 Linux 单行命令，希望对你有用。

  1. sudo !!
    以 root 帐户执行上一条命令。

  2. python -m SimpleHTTPServer
    利用 Python 搭建一个简单的 Web 服务器，可通过 http://$HOSTNAME:8000 访问。

  3. :w !sudo tee %
    在 Vim 中无需权限保存编辑的文件。

  4. cd -
    更改到上一次访问的目录。

  5. ^foo^bar
    将上一条命令中的 foo 替换为 bar，并执行。

  6. cp filename{,.bak}
    快速备份或复制文件。

  7. mtr google.com
    traceroute + ping。

  8. !whatever:p
    搜索命令历史，但不执行。

  9. $ssh-copy-id user@host
    将 ssh keys 复制到 user@host 以启用无密码 SSH 登录。

  10. ffmpeg -f x11grab -s wxga -r 25 -i :0.0 -sameq /tmp/out.mpg
    把 Linux 桌面录制为视频。

多合一读卡器Windows7下出现Unknown Device错误的解决办法

发表于 2011-03-07 | 更新于 2017-11-08 | 分类于 技术随笔

多合一读卡器Windows7下出现Unknown Device错误的解决办

尝试了n多办法都不好用啊，Unknown Device Identifier也不行

最后禁用了两个不知道干什么的usb驱动，终于可以用了

解决办法：

右键我的电脑，属性 – 设备管理器 – 通用串行总线控制器（就是有很多usb驱动的那个）禁用 Standard Enhanced PCI to USB Host Controller

名字要是不完全一样就找差不多一样名字的也可以，不要卸载，卸载了会自动安装，禁用就可以，以防涉及到其他的再恢复。
