bo-blog插入YouTube的视频方法

发表于 2009-02-08 | 更新于 2017-11-08 | 分类于 技术随笔

今天想在博客里插入一个YouTube的视频，发现正常的方法不可以，播放不出来。后来发现只要稍稍的改动一下就可以了。

方法：

http://www.youtube.com/v/5u_lycWhWw0

一次惊心动魄的域名抢注经历

发表于 2009-02-06 | 更新于 2017-11-07 | 分类于 技术随笔

zhaojian.net这个域名停了近三个月了，想再重新注册下来，关注了好多天，终于给抢注下来了。毕竟以前的记录都是我写的，自己也不想丢掉这个域名。所以准备抢注，等了一天，在凌晨4点多的时候终于给注册下来了。

域名抢注就是一个域名在被别人注册之前，抢先将此域名以注册人的名义注册下来的行为。好的国际域名基本上已经被欧美等发达国家瓜分完毕。现在要得到一个好的国际域名，要么高价购买。要么抢注。每天都有大量的过期未续费的域名被踢出，这里边不乏许多的好域名。只要你比别人抢先一步，以自己的名义注册下来，那么它就是你的了。（当然你要记得续费，要不到期后同样会被踢出，很多好域名就是这样被踢出来的哦）那么，如何才能比别人抢先一步，抢注自己喜欢的域名呢？

　　一.如何查询过期之后被踢出的域名

　　推荐查询网址：

　　1. pool.com 你可以查到最近几天要踢出来的域名。

　　2.http://www.backorder.com.cn/

　　要删除域名的世界排名（www.alexa.com），pr值（google的重要性评估）等等，都可以在这里查询到。

　　要仔细查找哦，或许花上几十块钱注册个域名，每天就能给你带来成千上万的 ip。

　　3.http://www.kingnic.com/

　　注册就看自己的情况了，这几个网站都不便宜，你可以选择自己的比较便宜的，我的是在这注册的 朝阳数据中心

　　二.抢注域名的预备工作

　　换算成北京时间。例如 2-5 删除的域名，实际上是在北京时间 2-6 的凌晨 2点—4点删除，.org类型的域名，有少部分会比实际删除时间提前多达一天踢出来，这是特例。

精简 QQ2009 Beta 中文版 组件的方法

发表于 2009-02-06 | 更新于 2017-11-07 | 分类于 技术随笔

精简 QQ2009 Beta 中文版 插件的方法（可以去广告了）
我前几天写的那个QQ2009/TM2008插件管理的程序，在QQ2009 Beta上失效了（QQ2009 Beta不能启动）。今天看到官方的QQ2009 Beta了，研究了一下，找到破解的办法，那就是先下载一个英文版的QQ2009 Beta版，然后将英文版的QQ2009 Beta的bin目录下的所有文件复制到中文版的 QQ2009 Beta的bin目录下（替换文件）（启动后QQ2009仍然是中文的），然后就可以继续使用我的那个插件管理程序了（广告只是其中的一个插件）。不过这样做之后，有一个限制，音频视频、文件传输、QQ邮箱、QQ秀、远程协助这5个插件不能删除或者禁用，否则QQ2009 Beta就不能启动了。

最后经过排除法，终于找到真正的文件了，那就是bin目录下的 AppMisc.dll  和 ChatFrame.dll。将附件解压后得到 AppMisc.dll 和 ChatFrame.dll，覆盖 QQ2009 Beta 的 bin 目录下的同名文件，就可以继续用原来的方法精简插件了。

把 UltraEdit-32 恢复到初始默认配置

发表于 2009-02-06 | 更新于 2017-11-07 | 分类于 技术随笔

今天不小心把UltraEdit-32搞乱了，字体不管怎么设置，都回不去了，然后那些菜单也被弄得乱七糟八的，于是就想把这些设置都搞成默认的。

重装UltraEdit-32、清理注册表、结果发现都没有用，设置就是没有变化。

得出一个结论，电脑里面肯定有个固定的地方放着UltraEdit-32的配置信息。

可是这个地方在哪里呢？还好，终于找到了答案！

答案就在UltraEdit-32的帮助文档里面，原话如下： “可配置的菜单的数据默认贮存在 %APPDATA% 目录下的 IDM Comp\UltraEdit 中的 uedit32.mnu 文件中。(对于 UEStudio 这个文件在 UEStudio 目录下，而不是 UltraEdit 目录)。”

打开“%APPDATA%” （不包括引号、复制到地址栏，然后回车就出来了），然后找到打开“IDM Comp” 文件夹，将里面的文件夹“UltraEdit”整个给删除了，然后再打开 UltraEdit-32 就会发现UltraEdit-32就变成刚装完时候的样子了！

端午节为什么吃粽子？

发表于 2008-05-14 | 更新于 2017-11-07 | 分类于 生活随笔

一直弄不明白端午节为什么要吃粽子？后来经过长时间的考证与调查，我发现了一个不为人知的秘密。

战国末期，群雄争霸，在楚国的一个比较贫穷的家庭里出生了一个小孩，因为这孩子很有才而生在这个家庭环境有点委屈，就他取了个名字叫屈原。屈原果然是个天才，在楚怀王十一年的时候，由文学侍臣晋升为左徒。这职务相当高，仅次于楚国最高行政长官令尹，相当于后来的副宰相。但是虽然有才，屈原的家教却很严厉，不能像其他小孩一样乱要东西，非分的请求爸爸妈妈是不会答应的。有一次，屈原出去溜达，遇见卖粽子的了，就非常想吃，馋的都不行了。后来回家就跟他妈妈要粽子吃，他妈妈就是不给他买。不管屈原怎么闹，怎么的求，就是不给买。后来屈原感觉活着也没什么意思了，连吃个粽子这个小的愿望都满足不了，一来气就跳河淹死了。由于当时的救援设备很落后，人掉到河里基本上就没救了，有人听到屈原一边挣扎一边还喊着，下辈子我死也要吃粽子。然后人们就赶快的往河里扔粽子，满足屈原的遗愿，让他死的瞑目。再后来，人们为了不让自己家的小孩子跳河就在端午节那天包粽子给小孩吃，于是就有了端午节。

呵呵，这是洗澡的时候远哥想起来的，后来他写在博客上了，我给copy过来修改了一下。

人生没有 Ctrl+z !

发表于 2008-04-13 | 更新于 2017-11-08 | 分类于 生活随笔

使用过电脑的朋友或多或少地都使用过Ctrl+z这个功能。Ctrl+z在电脑里的作用是撤消上一个操作，恢复到以前的某一步。和电脑相比的话，生活中可以按下Ctrl+z的地方实在是太少了，能够撤消的都是无关紧要的，不能撤消的反而都是举足轻重的。

“覆水难收、亡羊补牢”就是指的这个。时间不可以Ctrl+z，人情不可以Ctrl+z，说过的话，留过的言都不可以Ctrl+z。我们的生活有多少事可以重做？当一件事已经无可挽回的时候，我们应该做的，也是只能做的是弥补错误和承担责任！所以认真对待每一件事，人生没有Ctrl+z！

如今，肯能我需要撤销的太多太多了。我想忘掉一些记忆，或者不想面对那些发生的事情。我也想让自己回到过去，那个不会在乎别人什么感受的我，只顾着自己潇洒，不会去想别人。那样并没有什么不好，那样我至少可以自己过得舒坦。可现在一心为别人着想，却没有人会在意我的感受。因为这，我失去的太多了，得到的太少了。

一直对别人讲人要懂得什么时候珍惜，什么时候放弃。要知道什么时候该做什么事，我想现在我就是那样吧。呵呵，自私一点，为自己想想，自己能得到的可能更多。不要等付出过后悔了，才感觉当初根本没有那个必要！

个人站长必须记住的10个关键词

发表于 2008-03-19 | 更新于 2017-11-08 | 分类于 技术随笔

第一个关键词：发现，就是说要善于寻找和发现机会。机不可失，失不再来，机会在哪里？中国有句俗语，叫“缺什么，补什么”，也就是看互联网上还需要什么，今天需要什么，将来需要什么。

第二个关键词：量力，就是要根据自己的实力对自己的网站进行准确的定位。限于资金、技术、人员等诸多因素，个人网站定位，不要大而全，而要小而专，专而精，抓住关键，集中力量于“一个点”。

第三个关键词：第一，就是网站要有竞争优势，能在同类网站中处于领先地位。技术没有优势，我能做，别人也能做。规模，没有优势，没有强大的资金支持，短时间内很难做得很大。唯一的优势就是：网站的管理者。网站能够短时间内复制，但一个人的经历、经验、能力、思想、思维方式、社会关系是不能复制的，人才是不能复制的。要相信自己，能把网站做好、管理好、经营好。

第四个关键词：独特，就是网站要有创新，与众不同，才有魅力。

第五个关键词：营利，就是网站要有稳定可靠的营利模式。

第六个关键词：诚信，就是网站信息要真实可靠，要保证网站内容的真实性。

第七个关键词：共赢，就是要善于与别人合作，互惠互利，共同发展。

第八个关键词：长远，就是网站发展要有可持续性，避免急功近利的短期行为。要正确摆布网站流量、文章数量、网站质量、用户体验四者的关系，坚持质量第一的原则，以访客感觉为第一目标，以干成一个可以长久发展事业为方向，做到稳扎稳打，步步为营，循序渐进，分阶段、有计划，稳步发展。

第九个关键词：团队。就是网站的发展要有一批志同道合、强力互补的人员队伍。

第十个关键词：执著，就是做网站要不怕困难，坚持到底。网站发展过程中可能遇到资金紧张、对手竞争、黑客攻击等各种各样的困难。但我相信，有一种执著的精神，有不畏艰难的勇气，对于创新，敢于探索，敢于实践，最终一定会取得发展和成功。

以上十个关键词，是干成一番事业不可缺少的，也是作为一个个人网站站长所必须应该想到和做到的。我认为，对于个人站长，无论做一个什么样的网站，只要你真正悟“透”了这十个关键词的内涵，并真真正正地那样做了，做好了，那么，一切，都有可能成为最有可能的可能。

非主流网站面临的严峻问题--Firefox推介和Picasa推介价格降低

发表于 2008-03-15 | 更新于 2017-11-08 | 分类于 生活随笔

一直以来以推介Picasa为主来赚取广告费的各种非主流图片网站现在要面临一个严峻的问题了。

Google曾经在一个月前通知将要降低来自中国大陆的Firefox推介的价格，目前看来这项措施已经实施了，而且价格降低的程度之大令人惊讶。

Firefox推介的运作方式是：当网站推介使用带有Google工具栏的Firefox的用户首次运行Firefox时，网站管理员可以获得一定的收入，具体金额视用户所在位置而定。推介对象应为从未安装过Firefox的Windows用户才可以获得该收入。

目前FireFox推介和Picasa推介的价格降低到每个中国大陆的转换用户为0.01美元，而先前的价格为每个转换0.1美元，降低了十倍之多。相比之下，美国用户的转换价格依旧为1美元，目前中国和美国的价格相差竟然高达100倍。

据分析，Google如此降低中国区域的价格可能是因为这些区域推广效果不好，Google不想继续花冤枉钱，所以就干脆将其价格降到一个极低的价格，以避免自己的损失。

在目前的这种价格体系下，中国用户做FireFox推介已经没什么意思了，因为根本就赚不到钱了。放着FireFox推介的还是改放其他广告合算一些。
