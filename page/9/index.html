16款Chrome扩展插件以及谷歌浏览器插件扩展中心无法访问解决办法

发表于 2010-01-01 | 更新于 2017-11-07 | 分类于 技术随笔

1.PDF Download – 将网页转换为PDF文档

当你在使用Chrome浏览网页时，PDF Download允许你将网页转化成高质量的PDF文档。安装方法也很简单，打开PDF Download官方页面，然后将 “Save Page as PDF”拖到浏览器的标签栏，当需要将某一网页转换为PDF文档时，点击“Save Page as PDF”，稍等片刻即可免费获得高质量的PDF文档了。

2.Google Chrome Backup

Google Chrome Backup是一款允许用户创建、恢复、备份、管理个人资料的Chrome扩展。它能管理的个人资料包括浏览历史、书签等个人设置信息。基本上任何信息都能一键完成。

3.TabsLock

让 大写切换键具备快速新建Chrome标签页功能。TabsLock允许Chrome用户使用Caps Lock键快速建立Chrome标签页，即使Chrome是在不运行的情况下。如果想停止运行TabsLock，右键点击屏幕右下角的图标，执行Stop TabsLock即可。

4.ChromePass

ChromePass是一个很“不起眼”的密码恢复工具。通过它，你可以将使用户能够轻松地查看用户名和密码。每当输入一个密码，你就获得以下信息：原始网址，登录帐号的URL，使用者名称字段，密码字段，使用者名称，密码，以及登录时间。

5.ChromeMailer

Google Chrome更新频率很快，且几乎每次更新都会带给用户惊喜。但Google官方一直没有开发出支持“在Chrome中点击页面上的 mailto:邮件地址时，直接调用Gmail来发送邮件”的功能，（现在Chrome默认的是，当在Chrome 中点击页面上的 mailto:邮件地址时，默认的是调用系统中安装的电邮客户端来发送邮件）。而ChromeMailer即可实现直接通过Gmail发送邮件的功能。

6.GMail This! Bookmarklet{m=)

GmailThis!是我用过的最多的浏览器工具栏书签。使用GmailThis!，你无需访问mail.google.com即可撰写博客。当你把GmailThis!的链接添加到浏览器工具栏后，收发邮件就变得简单的多了。使用方法如图所示。

7. Cleeki

该扩展实现了类似IE8中的Accelerators加速器功能，当高亮选中文本后会出现提示选项，用户可执行查找、翻译、搜索等相关操作，此外Cleeki可以将IE8加速器内容加载到Chrome。

8.AdSweep

一个Chrome浏览器去广告插件。其基于JavaScript，能够针对一些网页上的广告自动进行CSS调整，以达到隐藏广告的目的.

9. LastPass

LastPass 之前就是 Firefox 上一款出色的密码管理应用扩展，现在，开发团队发布了 Chrome 版本的测试版，提供更为安全方便的密码管理应用。

10. Gmail Checker

Gmail Checker的作用就是在浏览器地址栏的一个按钮上实时的显示你Gmail里有多少未读邮件，点击这个按钮就可以进入Gmail查阅这些未读邮件，很好理解;

11. Google Reader Checker

顾名思义，就是检测你Google阅读器里有多少未读的条目，和Gmail Checker一样，也是显示在地址栏右边;

12. Google Tasks

这个小插件就是让你在Chrome浏览器内部直接查看Google Tasks的，在Chrome浏览器右下角有个Tasks面板，很好用的，可以展开或隐藏。

13. gPDF

这个就是在Chrome浏览器中直接阅读PDF文件的小扩展，在《在Chrome中阅读PDF的扩展：gPDF》这篇文章中已经有详细的介绍了。

14. Google Wave Checker

还是顾名思义就知道，这个小扩展是用来检测你有多少未读的Google Wave条目的，不过这个是显示在Chrome浏览器的左下角工具栏里，你也可以将它放到书签栏中，方法请参考Chrome迷论坛。

使用chrome扩展的方法：目前chrome被墙，需要翻墙，下面介绍最简单的方法

Google推出Chrome扩展页面后有些朋友可能访问不了（请不要讨论原因，众所周知）。现时可以通过修改hosts文件的方法来解决。

新建立一个文本文档，然后输入如下内容：

attrib -s -r C:\WINDOWS\system32\drivers\etc\hosts
call notepad.exe C:\WINDOWS\system32\drivers\etc\hosts
attrib +r +s C:\WINDOWS\system32\drivers\etc\hosts
exit

然后将该文件改名为：a.bat

运行创建的a.bat文件，在弹出的记事本编辑器中，在最下边新建两行内容输入：

74.125.39.99 chrome.google.com
74.125.39.99 clients2.google.com

保存退出。

AV-Test揭晓2009年度最佳杀软

发表于 2009-12-27 | 更新于 2017-11-07 | 分类于 技术随笔

临近岁末，著名安全软件评测机构av-comparative发布了2009年年终杀软测试总结报告。报告全面盘点了一年来所有的测试结果，并对每款参加测试产品展开评定，授予表现出色的部分产品以最高荣誉。

Av-comparative的入选资格非常苛刻，对测试对像品质的要求相当高，今年的16款测试产品是：Avast、AVG、 BitDefender、eScan、ESET NOD32、F-secure、G-Data、Kaspersky、McAfee、MSE、Kingsoft、Norman、Trustport、 Symantec、Sophos。

2009年最佳杀软产品：Symantec

作为最佳的杀软产品，必须具备以下条件：很高的病毒检测率（包括PUA潜在威胁）、高防护功能、较低的误报率、扫瞄速度、清除恶意程序、和良好的兼容性等八个方面。

评比结果：奖项前三名分别是——金奖Symantec、银奖Kaspersky、铜奖ESET。去年的冠军Avira跌出前三名，而Kaspersky仅以微弱劣势屈居第二，历史上Kaspersky和ESET近5年内均获得两次冠军头衔。除此以外，BitDefender和F-secure排在第四、五名。

具体测试结果排名：

汇集了八种分项测试的排名，其中Symantec、MSE、ESET非常抢眼。而中国的国产杀软金山Kingsoft在扫瞄速度和系统兼容性方面表现优异，摘得分项评比的第一和第二位。
Av-comparative对16款主流杀软的点评：

1、Avast！

与以往相比，Avast在检测率上有很大进步（特别是今年下半年），误报率也明显下降。新发布的Avast 5.0在用户界面上有很大改动，新增了不少防护功能，非常适合于家庭用户。

主要特点：高检测率、扫瞄速度快、免费版本。

2、AVG

AVG今年的表现不让人满意，但是它的口碑依然不错，期待明年AVG能取得好成绩。AVG套装中包括反病毒组件和网络安全组件。

主要特点：易于使用、免费版本和LinkScanner功能。

3、Avira

Avira是去年的冠军，同样具有极高的检测率和主动防御能力，但是误报率也较高，影响了整体成绩，新版Avira将在新增行为监考功能，弥补此项不足。

主要特点：高检测率、良好的兼容性、免费版本。

4、BitDefender

BitDefender整体表现中规中距。

主要特色：检测率、低误报、智能探索、清除恶意软件。

5、eScan

eScan是一款基于BitDefender的多引擎杀软产品，性能与其比较类似。

主要特色：高检测率、清除恶意软件能力。

6、ESET NOD32

NOD 32今年成绩还算不错，取得铜奖，历史上两次获得冠军。

主要特色：高检测率、智能防护、系统兼容性。
　　
7、F-secure

F-secure也是一款多引擎的安全产品。性能提升显著，在国外用户中占有率较高。2010版包括反钓鱼攻击和全新的使用界面。

主要特色：高检测率、清除恶意软件、友好的操作界面。

8、G-Data

G-Data是基于Avast和BitDefender的联合引擎，扫瞄能力出众。

主要特色：高检测率、联合引擎、易于上手。

9、Kaspersky

Kaspersky是今年的亚军，历史也多次获得较高评价。

主要特色：智能检测、高病毒检测率、多种保护功能。

10、Kingsoft金山

金山Kingsoft是第一款挑战AV-comparative测试的中国杀毒软件，它强大的清除病毒和恶意程序的能力在同行业中处于领先地位，但误报率问题也值得注意。

主要特色：扫瞄速度快、兼容性好、用户界面人性化。

11、McAfee

麦克菲是老牌安全软件厂商，今年延续了其一贯的传统，新版提供离线防护功能，用户界面也得到了很大改进。

主要特色：高恶意软件检测率、高PUA检测率、网站助手。

12、Microsoft Security Essential

微软今年推出的安全套装MSE可谓是进步最快的一款产品。获得三个单项奖提名。

主要特色：主动防护、低误报率、清除病毒能力、免费版本。

13、Norman

欧洲老牌杀软Norman表现不佳，在八项测试中只通过了其中的四项。

主要特色：支持旧版本系统（Win 95等）、屏保模式扫瞄、简单的用户界面。

14、Sophos

英国杀软Sophos拥有独特的HIPS系统，用于检测潜在危险适合于企业级用户，故而在av-comparative中表现平平。

主要特色：易于上手、针对企业用户、HIPS。

15、Symantec

从今年开始，在保持传统优势的同时，Norton一直关注其占用过多资源的问题，并取得了明显的进步。新版界面华丽，功能全面，获得AV-Test的最高奖项。

主要特色：极高的检测率、良好的兼容性、功能完善。

16、Trustport

Trustport同样是整合多种引擎的安全产品，用户可根据需要选择使用，默认是AVG和BitDefender的扫瞄引擎。缺点是扫瞄速度较慢、误报率高。

主要特色：极高的检测率、极高的PUA检测率、选择式引擎。

社区与平台服务提供商PHPWIND被支付宝收购

发表于 2009-12-27 | 更新于 2017-11-08 | 分类于 技术随笔

专注于网站构架设计的社区与平台服务提供商PHPWIND近日向腾讯科技确认，已隶属于阿里巴巴集团，目前是支付宝旗下公司，CEO直接向支付宝总裁邵晓锋汇报工作。

PHPWIND副总裁肖睿哲向腾讯科技表示，事实上早在一年前此事已经成形。但阿里集团方面不希望此事影响到PHPWIND的日常运维，所以一直很低调。

肖睿哲说：“我们目前有100多名员工，95%都是做技术的。这个时候突然增加一部分负责外联、媒介等等事物的人，我们没有这个驾驭能力。”

事实上，早在今年10月17日中国地方网站发展论坛上，邵晓峰就承认阿里巴巴已将PHPWind纳入旗下。在肖睿哲看来，阿里巴巴这一举动是为了在全国普及发达的电子商务基础。

肖睿哲说，PHPWIND正在帮助各地区的本地门户网站进行升级和改造，并向他们提供成熟的广告模式，“目前针对各小区住户的房产、汽车广告很热，而且淘宝平台那么多卖家，本身每年就有好几亿的广告资源无法消化。”

此外，根据各地区本地门户网站的需求，PHPWIND还提供淘宝和支付宝的接口，方便地方门户网站发展电子商务和社区互动。肖睿哲表示目前PHPWIND的工作还没有盈利，主要是在宏观层面把成熟的电子商务和互联网门户理论拓展到更大的平台。

“在PHPWIND的帮助下，涌现出了一批优秀的地方门户网站。”肖睿哲表示。

  • 增加了 TO_SECONDS 函数
  • 新的 partition语法
  • Key caches are now supported for indexes on partitioned MyISAM tables, using the CACHE INDEX and LOAD INDEX INTO CACHE statements
    这个版本还不适合用在正式环境。想想以后也许可以用mysql mmm + mysql 5.5提供可用性更好的服务，前提是它已经足够稳定，或者是percona版本？

中国域名监管机构CNNIC接连发出加强域名注册信息审核工作的公告，而最新公告规定，从2009年12月14日上午9时起，个人用户没有资格进行域名注册，因此许多站长，尤其是bloggers都把目光投向了海外域名。

在国外购买域名理由：

一，可以得到他们提供很多的增值服务。

二，可以自由来回转出，转入，方便过户。

三，都是ICANN的顶级代理商，不像国内多数的那些，今天注册，明天跑路，找不到人。

不过国外注册也是有缺点的，就是支付不方便，客服沟通难。国内的部分地区ICP可能会对一些域名进行DNS劫持，这个比较高级点。

不在国内注册域名的理由：

一.安全保证程度不高，他们会屈服于官方压力而不给你解析，甚至收回你的域名。

二解析速度特别慢，国外的几分钟能解析好的，他们要几个小时，特别是.cn的域名（我碰到的最长竟然需要一天才能解析的完）。

三.据我所知到现在国内还没有任何一家域名注册商直接提供域名转移密码，都会让你办很麻烦的手续去转移这个域名。

四.功能上比国外的差很多，服务态度也很差。
