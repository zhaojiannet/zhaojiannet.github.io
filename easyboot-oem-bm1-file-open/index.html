系统封装时EASYBOOT制作的oem.BM1文件打开方法
2024-10-30 10:35:21 # 技术

用uedit32或emedit或者其他什么能用十六进制读文件的程序打开它,最前面是45 42,把它改成42 4D,然后改后缀为BMP就可以了

改完后再用PS或者FW 180度旋转—再水平旋转一下,再把亮度和对比度调一下.
